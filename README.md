# 🇮🇳 Indian Retail GenAI Support System

## 🚀 Overview

This is a GenAI-based customer support system for Indian retail platforms.

Built using:

* LangChain
* LangGraph

## 💡 Features

* Order tracking
* FAQ responses
* Complaint handling
* Hinglish query support

## 🧠 Architecture

User Query → LangGraph → LangChain → Response

## 📌 Future Scope

* Add RAG
* Add CrewAI agents
* Add UI (Streamlit)
* Add API integration
